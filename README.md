# Intel_Image_Classification

## Dataset

Get the Dataset from Kaggle [Intel Image Classification](https://www.kaggle.com/puneet6060/intel-image-classification)


## Description

In this dataset there are 6 different classes.

1. Buildings
2. Glacier
3. Sea
4. Forest
5. Street
6. Mountain
